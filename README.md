# 🛒 ShopEase Admin Panel

A simple and responsive Admin Panel for managing products in the ShopEase eCommerce application.

---

## 🚀 Features

- ➕ Add New Products
- 💰 Product Price Management
- 🖼️ Product Image URL Support
- 📂 Category Selection
- 📋 View All Products
- 🎨 Clean User Interface
- ⚡ Fast and Lightweight

---

## 📁 Project Structure

```bash
ShopEase/
│
├── index.html          # Store Frontend
├── admin.html          # Admin Dashboard
├── style.css           # Styling File
├── admin.js            # Admin Functions
├── script.js           # Store Logic
└── README.md
```

---

## 🛠️ Technologies Used

- HTML5

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/shopease.git
```

### Open Project

```bash
cd shopease
```

### Run Locally

Simply open:

```bash
index.html
```

or

```bash
admin.html
```

in your browser.
